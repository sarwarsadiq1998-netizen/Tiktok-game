# TikTok Interactive Game

لعبة تفاعلية للبث المباشر على تيك توك.

## الميزات
- عداد اللايكات الفوري وأكثر 5 مشاهدين تفاعلاً بألوان مميزة.
- منافسة أسبوعية بين الستريمرز.
- نظام اشتراك بقيمة 2$ عبر الهدايا أو البطاقة.
- موجات حارة (Heat Streaks) تضاعف النقاط.
- مسابقات سريعة في البث.
- لوحة تحكم للستريمر.
- شاشة حائط الشهرة في نهاية الأسبوع.

## التشغيل السريع
1. أنشئ ملف `.env` من `.env.example` واملأ البيانات.
2. نفذ `npm install`.
3. شغّل PostgreSQL و Redis.
4. نفذ `npm start`.

## الوصول
- تراكب OBS: `http://localhost:3000/overlay`
- لوحة التحكم: `http://localhost:3000/admin`
- شاشة الشهرة: `http://localhost:3000/wall`
